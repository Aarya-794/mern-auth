# Server-side Dependencies
Install these first to run the backend server:
npm install express mongoose dotenv jsonwebtoken bcryptjs nodemailer cors cookie-parser

# Dependency Info

# express
# Used to create the server and API routes

# mongoose
# Connects and communicates with MongoDB

# dotenv
# Loads environment variables from .env file

# jsonwebtoken
# Used to generate and verify JWT tokens for user authentication

# bcryptjs
# Used to hash and compare passwords securely

# nodemailer
# Sends emails (e.g., for account verification or password reset)

# cors
# Enables requests from frontend (React) to backend (Node.js)

# cookie-parser
# Parses cookies from incoming requests (e.g., to read JWT from cookies)
